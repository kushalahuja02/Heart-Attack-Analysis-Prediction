Training the model - Gaussian Naive Bayes¶

We used the Gaussian Naive Bayes model (GaussianNB) because it's often a good choice when features are continuous or have a normal-like distribution. For features that are categorical, it's better to use the categorical or multinomial Naive Bayes models (CategoricalNB or MultinomialNB) instead.

from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import GaussianNB

from sklearn.preprocessing import StandardScaler

from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import GaussianNB
from sklearn.metrics import classification_report, confusion_matrix
from sklearn.preprocessing import StandardScaler, KBinsDiscretizer
from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer
from sklearn.model_selection import GridSearchCV

# Define features and target
X2 = df_cleaned.drop(columns='target')
y2 = df_cleaned['target']

# Define a preprocessor to bin 'age'
preprocessor = ColumnTransformer(
    transformers=[
        ('age', KBinsDiscretizer(n_bins=3, encode='ordinal', strategy='uniform'), ['age'])],
    remainder='passthrough')

# Split the data
X2_train, X2_test, y2_train, y2_test = train_test_split(X2, y2, test_size=0.2, random_state=42)

# Define the model
gnb = Pipeline(steps=[('preprocessor', preprocessor),
                      ('scaler', StandardScaler()),
                      ('classifier', GaussianNB())])

# Define the hyperparameters to tune
param_grid = {'classifier__var_smoothing': np.logspace(0,-9, num=100)}

# Use GridSearchCV to find the best hyperparameters
grid_search = GridSearchCV(gnb, param_grid, cv=5)

# Train the model
grid_search.fit(X2_train, y2_train)

# Print the best parameters found
print("Best parameters for Model 2:")
print(grid_search.best_params_)

# Use the best model to make predictions
best_gnb = grid_search.best_estimator_
y2_pred = best_gnb.predict(X2_test)

# Print the classification report and confusion matrix
print("Model 2 Results:")
print(classification_report(y2_test, y2_pred))
print(confusion_matrix(y2_test, y2_pred))

Model 2 has been able to correctly predict 26 out of 29 instances of class '0' and 28 out of 32 instances of class '1', which are indeed better results compared to the baseline model.

However, it's worth noting that although the model's accuracy is fairly high, it's always important to look beyond accuracy, especially in cases where the classes are imbalanced. Precision, recall, and the F1-score give us a more comprehensive view of the model's performance, and in this case, they all show good results.
